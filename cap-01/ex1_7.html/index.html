Calculando o valor do jantar e utilizando o metodo toFixed(2)

O método "toFixed(2)" é adicionada ás variáveis de saída. Ele serve para indicar que o valor a ser apresentado deve conter 2 casas decimais

        const ValorJantar = Number(prompt("Quantos deu o valor do jantar?"))
        const Taxagarcom= ValorJantar*0.10
        const total= Taxagarcom+ValorJantar
        alert(`Taxa Garçom R$: ${Taxagarcom.toFixed(2)}\nTotal do Jantar R$: ${total.toFixed(2)}`)
