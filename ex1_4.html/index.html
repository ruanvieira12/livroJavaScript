Declarar variáveis e exibir seus tipos a partir do comando typeof

Exibir uma variável que não recebeu uma atribuição de valor vai gerar uma saída "undefined".


                const fruta= "Banana"
                const preco= 3.50
                const levar= true
                let novoValor
                console.log(typeof fruta) // string
                console.log(typeof preco) // Number
                console.log(typeof levar) // Boolean
                console.log(typeof novoValor) //undefined
